# UY Import Ops Dashboard

Dashboard de investigación de productos para importación + marketing automatizado.

## Deploy en Render.com

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/TU-USUARIO/uy-import-ops)

## Características

- Product Hunter con 100+ productos precalibrados
- Calculadora de costos de importación (CIF, aranceles, IVA, IMPO)
- Estrategias de pricing (6 modelos)
- Marketing: Listing ML, Social, Email, Estrategia completa
- Diseño gráfico con IA (banner, mockup, flyer, hero) usando producto real compuesto
- Campañas guardables manualmente

## Stack

- Python 3.11 + FastAPI
- SQLite
- Cloudflare Workers AI (image generation)
- Frontend: HTML + Tailwind (single-file)
